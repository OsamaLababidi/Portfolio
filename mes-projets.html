Jeu Puissance 4

Ce projet s'est déroulé sur une période d'un mois, avec une équipe de cinq membres. Notre objectif était de concevoir un jeu Puissance 4 avec une interface graphique en utilisant le langage de programmation Python. J'ai été responsable du développement de l'interface graphique du jeu. Par la suite, nous avons réalisé une présentation orale pour exposer notre jeu et démontrer son bon fonctionnement. Nous avons utilisé diverses applications, notamment GitHub, pour faciliter la communication et le partage du code entre les membres de l'équipe. Le projet a été mené à bien dans les délais impartis.

Calculatrice

Ce projet a été réalisé sur une période d'une semaine en binôme. Notre objectif était de développer une calculatrice dotée d'une interface graphique, en utilisant exclusivement les concepts de pile, file et liste en Python. Afin d'atteindre cet objectif, chaque membre de l'équipe s'est concentré sur une partie spécifique, et nous avons réparti équitablement les tâches. Ce projet m'a permis d'approfondir ma compréhension des notions techniques du langage Python.

Mot De Passe

Ce projet a été réalisé sur une période d'une semaine en binôme. Notre objectif était de développer un générateur de mots de passe afin de permettre aux utilisateurs de créer des mots de passe sécurisés. Le générateur de mots de passe comportait des spécificités telles qu'un nombre minimal de caractères, l'inclusion de caractères spéciaux et de majuscules, entre autres. Pour mener à bien ce projet, nous avons réparti équitablement les tâches entre nous. Ce projet m'a permis de comprendre les bases du langage Python, car j'étais novice dans ce langage à l'époque.

Code Barre

Ce projet a été réalisé sur une période de 2 mois, avec une équipe de 5 personnes. Notre objectif principal était de développer un jeu en 2D qui permettrait à l'utilisateur d'explorer un labyrinthe avec un personnage et d'interagir avec divers éléments rencontrés. Pour atteindre cet objectif, nous avons utilisé plusieurs langages de programmation, principalement Python, et avons mis en place une base de données pour stocker les éléments du labyrinthe. Mon rôle principal consistait à travailler sur la partie graphique du jeu, tout en apportant mon aide aux autres membres de l'équipe en cas de difficultés. Ce projet a été une occasion pour moi d'améliorer mes compétences en matière de collaboration et de découvrir les exigences d'un véritable projet.
