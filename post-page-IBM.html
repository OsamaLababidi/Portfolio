Projet :

SAE1.05-06 - Développement d'un site web

Objectif :

Réaliser un site web permettant aux élèves de 3ème de s'informer sur IBM en vue d'un éventuel stage.

Description :

Ce projet consistait à recueillir les besoins du client et à rechercher des informations pertinentes sur IBM, en mettant un accent particulier sur l'impact écologique de l'entreprise, ses objectifs et les mesures mises en place pour les atteindre.

Réalisation :

Dans le cadre de ce projet, nous avons conçu un site web permettant aux élèves de 3ème de s'informer sur IBM en vue d'une éventuelle candidature à un stage. Pour atteindre cet objectif, nous avons commencé par recueillir les besoins du client, puis nous avons recherché les informations clés concernant IBM. J'ai notamment axé mes recherches sur l'impact écologique d'IBM, ses objectifs et les moyens mis en place par l'entreprise pour les atteindre. Ensuite, nous avons élaboré une maquette préliminaire du site web à l'aide de l'application Whimsical, ce qui nous a donné un premier aperçu visuel. Une fois la maquette réalisée, nous avons entamé le développement du site web. Pour faciliter la communication et l'échange de code entre les membres de l'équipe, nous avons utilisé GitHub, qui nous permettait de collaborer en temps réel. Nous avons également utilisé Discord pour les échanges verbaux. Pendant la phase de développement, nous avons réparti équitablement les tâches, et j'ai été chargé de créer la section environnement du site web. Nous avons réussi à finaliser le site web dans les délais, mais nous avons tout de même été déçus pour plusieurs raisons. Tout d'abord, lors de nos recherches, de nombreuses informations manquaient de sources fiables, ce qui a entraîné une certaine incertitude concernant la fiabilité de nos données. En outre, nous avons rencontré plusieurs problèmes techniques lors de la création du site web, notamment des désaccords au sein de l'équipe et des difficultés de codage, ce qui a retardé notre progression et mis à mal notre projet.

Bilan :

Ce projet nous a permis d'améliorer de nombreuses compétences, notamment des compétences comportementales telles que l'autonomie, l'adaptabilité, le travail d'équipe et la flexibilité.Il m'a permit aussi de beaucoup m'améliorer sur le langage de programmation html et css.
